Secret Weight Guess

Set a hidden weight, let players submit encrypted guesses, and reveal how close they really are — without exposing the truth.

1️⃣ Set Encrypted Weight

Encrypt the true weight of the object (e.g. in grams). Players only see feedback levels.

2️⃣ Submit Encrypted Guess

Encrypt your guess; the contract will homomorphically measure how close you are. 

—

3️⃣ Make Public & Decrypt Feedback

Make your last feedback public and decrypt it via Relayer.
